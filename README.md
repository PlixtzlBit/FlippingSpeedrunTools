# If you already have mods installed/Have the correct version of BepInEx for this game, you can skip step 1
https://builds.bepinex.dev/projects/bepinex_be/755/BepInEx-Unity.IL2CPP-win-x64-6.0.0-be.755%2B3fab71a.zip <-- This link will have a .zip file

1. Download and extract it into Flipping Is Hard (NOT Flipping is Hard_Data), run the game once, wait a few minutes and when you see the title screen exit. (When you extract it, if it makes a folder that starts with BepInEx.Il2CPP.Win64 or something just take everything out of that folder and into the Flipping is Hard folder)

2. Then go into BepInEx > Plugins > and make a new folder named <Insert Mod name> then put the .DLL file inside there. If the mod uploader includes a zip file most likely it will have the folder for you, please follow any extra instructions from the uploader (Such as where to extract it!)

3. If you installed a mod already but want to update it just remove the old .DLL file, from the same <Insert Mod Name> folder you created (Or add the new zip file from the uploader)

4. Run the game again and you will have the mod you wanted (hopefully).

Important: If it doesn't work, then if you don't see a black window appear before the game then you didn't put BepInEx in properly. If you do but the mod doesn't work, check that you put the mod in the right place (Flipping Is Hard > BepInEx > Plugins > <Insert Mod Name>) 
All mods should work with Demo and Release unless there are major differences between them (Player, Menu or Map is reworked)
